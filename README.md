# ✨ About

Dzinr is a Progressive Web App (PWA) built for designers to:

- Upload their creative work.
- Import existing portfolios.
- Receive anonymous visual feedback.
- Discover high-quality design inspiration.
- Build a better portfolio through real community interaction.

Unlike traditional social media platforms, Dzinr focuses on **design quality** rather than follower counts.

The platform encourages thoughtful curation through simple interactions:

- 👈 Swipe Left → Not relevant for me
- 👉 Swipe Right → I like this
- ⭐ Save → I would genuinely use this as inspiration

# 🎯 Design Philosophy

Dzinr is intentionally **not** another social media platform.

The product is designed around one question:

> "Would another designer genuinely use this as inspiration?"

Everything else follows from that principle.

---

# 🎨 Design Language

Core principles:

- Calm
- Premium
- Editorial
- Minimal
- Purposeful
- Creator First

The interface stays out of the way so the design work becomes the hero.
